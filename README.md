# AI Resume Matcher

## 📌 Description
AI Resume Matcher is a web application that analyzes resumes and job descriptions to calculate a matching score and suggest improvements. It helps candidates optimize their resumes and assists recruiters in shortlisting candidates efficiently.

---

## 🚀 Features
- Resume analysis using AI
- Job description matching
- Match score calculation
- Suggestions for resume improvement

---

## 🛠️ Tech Stack
- TypeScript
- Vite
- Node.js
- Gemini API (AI processing)
